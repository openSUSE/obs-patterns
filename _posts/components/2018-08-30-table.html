--- layout: post title: Table category: components description: A `%div` container with the CSS class `.table-responsive` makes the table responsive. The width of columns can be adjusted by using the [Bootstrap's sizing classes](https://getbootstrap.com/docs/4.1/utilities/sizing/) on the `%th` tags ([Bootstrap's grid system](https://getbootstrap.com/docs/4.1/layout/grid/) doesn't work for tables in Chrome). Additionally, we are using [DataTable](https://datatables.net/examples/styling/bootstrap4)'s [responsive](https://datatables.net/extensions/responsive/) and style them with striped rows (`.table-striped`) and borders (`.table-bordered`).
